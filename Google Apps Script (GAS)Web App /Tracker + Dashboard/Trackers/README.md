# ApexFlow

ApexFlow is a high-performance daily task tracker designed for professional efficiency. This repository contains the modular architecture of the tracker, featuring native CSS and a premium design aesthetic.

## Versions

### Version 1
Initial functional tracker version.

### Version 2
Modular version with refactored HTML, CSS, and JS components. This version eliminates external frameworks like Tailwind CSS in favor of a custom, high-performance native CSS architecture.

## Key Features
- **Modular Design**: Separate components for structure, styling, and logic.
- **Premium Aesthetics**: Rich, modern UI with smooth transitions and vibrant nocturnal themes.
- **Performance Focused**: Optimized with native CSS for minimal overhead.
- **Daily Task Management**: Streamlined flow for tracking tasks and productivity.

## Getting Started
To view the tracker, open the `index.html` file within the respective version folder.
